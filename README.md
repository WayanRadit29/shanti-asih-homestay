# Shanti Asih Homestay Website

Shanti Asih Homestay Website merupakan mini project pada mata kuliah Pemrograman Web yang mensimulasikan sistem booking homestay berbasis frontend. Website ini dirancang untuk memberikan pengalaman pengguna dalam mengeksplorasi kamar dan melakukan proses pemesanan secara sederhana.

---

## 📌 Deskripsi

Website ini bertujuan untuk membantu calon tamu dalam:
- Melihat informasi homestay
- Mengeksplorasi daftar kamar
- Melihat detail kamar
- Melakukan simulasi booking secara online

Project ini berfokus pada **user experience sederhana, clean, dan mudah digunakan**, tanpa menggunakan backend.

---

## 🚀 Fitur Utama

### 🔎 Search Kamar
Pengguna dapat mencari dan memfilter kamar berdasarkan keyword secara real-time menggunakan JavaScript.

### 🛏 List & Detail Kamar
- Menampilkan daftar kamar dalam bentuk card
- Halaman detail berisi gambar, deskripsi, harga, dan fasilitas

### 📝 Form Booking dengan Validasi
- Input data pengguna (nama, kontak, tanggal, dll)
- Validasi real-time pada setiap field
- Pengecekan logika tanggal (check-in & check-out)
- Menampilkan hasil simulasi booking

### 📍 Informasi Lokasi
- Terintegrasi dengan Google Maps
- Memudahkan pengguna menemukan lokasi homestay

### 🎨 Responsive Design
- Tampilan menyesuaikan berbagai ukuran layar
- Dibangun menggunakan Bootstrap
