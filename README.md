# Shanti Asih Homestay Website

Shanti Asih Homestay Website adalah website sederhana untuk sistem booking homestay keluarga yang berlokasi di Ubud, Bali. Website ini dibuat sebagai mini project mata kuliah Pemrograman Web.

## Deskripsi

Website ini membantu calon tamu melihat informasi homestay, daftar kamar, detail kamar, fasilitas, lokasi, dan melakukan simulasi booking secara online.

## Lokasi

Jalan Gotama Selatan No. 25, Desa Padang Tegal, Kecamatan Ubud, Kabupaten Gianyar, Provinsi Bali, Indonesia.

## Fitur

- Home page
- Search kamar
- List kamar
- Detail kamar
- Form booking
- Kalender tanggal check-in dan check-out
- Informasi fasilitas homestay
- Responsive design menggunakan Bootstrap

## Fasilitas

- WiFi
- AC
- Kamar mandi dalam
- Kolam renang
- Yoga Sala

## Teknologi

- HTML
- CSS
- JavaScript
- Bootstrap

## Status

Project ini masih berupa simulasi frontend dan belum menggunakan backend atau database.
